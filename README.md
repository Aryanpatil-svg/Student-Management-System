# 🎓 Student Management System

A modern **Student Management System** built using **Python**, **Tkinter**, **Object-Oriented Programming (OOP)**, and **JSON File Handling**. The application provides a user-friendly graphical interface to manage student records efficiently with persistent data storage.

---

## ✨ Features

- ➕ Add New Student
- ✏️ Update Student Details
- ❌ Delete Student Record
- 🔍 Search Students Instantly
- 📋 View All Student Records
- 📊 Live Total Student Counter
- ✅ Unique Student ID Validation
- 💾 Automatic Data Storage using JSON
- 🎨 Modern Dark-Themed Tkinter GUI
- ⚡ Fast and Easy-to-Use Interface


